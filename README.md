video transcoder that shrinks file sizes using FFmpeg
with H.265/H.264, CRF quality control, and a drag-and-drop queue


download https://ffmpeg.org/download.html and add to PATH

tkinterdnd2  (optional – enables drag-and-drop)

pip install tkinterdnd2


Without it the app still works; use the **+ ADD FILES** button instead.

---

## Run

bash-
py shrink.py
